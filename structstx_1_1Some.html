Detailed Description

template<Swappable T>
struct stx::Some< T >

value-variant for Option<T> wrapping the contained value

Usage

Note that Some is only a value-forwarding type. It doesn't make copies of it's constructor arguments and only accepts r-values.

What does this mean?

For example, You can:

Option a = Some(vector{1, 2, 3, 4});

You can't:

vector<int> x {1, 2, 3, 4};
Option a = Some(x);

But, to explicitly make a take ownership, you will:

vector<int> x {1, 2, 3, 4};
Option a = Some(std::move(x));
